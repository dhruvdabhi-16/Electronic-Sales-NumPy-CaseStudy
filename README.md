# Electronic Sales Analysis using NumPy

## Project Overview
This project analyzes electronic sales data using only NumPy and Python.

## Dataset Information
- Total Records: 185,950
- Products: 19
- Cities: 9
- Period: January–December 2019

## Analysis Performed
- Dataset Overview
- Measures of Central Tendency
- Measures of Dispersion
- Correlation Analysis
- Monthly Sales Analysis
- Product Performance Analysis
- City-wise Sales Analysis
- Business Insights Dashboard

## Key Findings
- MacBook Pro Laptop generated the highest revenue.
- San Francisco was the top-performing city.
- December achieved the highest sales.
- Sales and Price showed a very strong positive correlation.

## Technologies Used
- Python
- NumPy
- CSV Module
- Jupyter Notebook
